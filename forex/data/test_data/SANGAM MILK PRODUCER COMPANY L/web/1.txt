Company sets Rs. 420-crore target this year
Sangam Milk Producer Company Limited, one of the leading milk and dairy products manufacturing company in the State, is planning to enter into international market.
The Guntur-based dairy which is producing and selling milk and its bi-products across the country for the last three and a half decades, has applied for export permission.
The company has set a target to make a turnover of Rs.420 crore this year.
Sangam Dairy Chairman and Ponnur MLA Dhulipalla Narendra Kumar said the governing body has decided to expand the business to all accessible areas and market the products in other countries.
We applied for export licence with the government. Once the company obtains permission we will enter into international market. Sangam Dairy is maintaining quality and good standards. We want to be No.1 in the commercial sector, said the chairman.
Rs.34 crore sanctioned
Under National Dairy Plan, the National Dairy Development Board (NDDB) has sanctioned Rs.34 crore loan for development of Sangam Dairy. Another Rs.8 crore has been sanctioned for construction of bulk milk cooling centres.
Products
The management has planned to set up more milk collection points and expand its business.
Besides Guntur district, Sangam dairy is having milk producers (farmers) in Prakasam, Chittoor, West Godavari and other neighbouring districts. We want to increase the capacity of the plant and pay good price to the farmers, said Mr. Narendra Kumar.
Sangam dairy is producing milk, curd, butter, ghee, skim milk powder, flavoured milk, doodh peda, kalakand, lassi, malai laddu, mysore pak and tea.
Very soon we will introduce new product Sangam Cream into the market, said the company chairman.

var getOrApply = function(value, context) {

}

"K": 1000

_renderShareCount = function(share, container) {
var $count = $(container).next('.cnt');
this._loadCount(share).done($.proxy(function(count) {
if(count) {
$count.text(count);
}
}, this));
}
_loadCount = function(share) {
var deferred = $.Deferred();
var countUrl = this._getCountUrl(share);
if(!countUrl) {
return deferred.resolve(0).promise();
}
var handleSuccess = $.proxy(function(response) {
deferred.resolve(this._getCountValue(response, share));
}, this);
$.getJSON(countUrl).done(handleSuccess)
.fail(function() {
$.get(countUrl).done(handleSuccess)
.fail(function() {
deferred.resolve(0);
});
});
return deferred.promise();
}
_getShareUrl = function(share) {
var shareUrl = getOrApply(share.shareUrl, share);
return this._formatShareUrl(shareUrl, share);
}
_getCountUrl = function(share) {

return this._formatShareUrl(countUrl, share);
}
_getCountValue = function(response, share) {
var count = ($.isFunction(share.getCount) ? share.getCount(response) : response) || 0;
return (typeof count === "string") ? count : this._formatNumber(count);
}
_formatNumber = function(number) {
$.each(MEASURES, function(letter, value) {
if(number >= value) {
number = parseFloat((number / value).toFixed(2)) + letter;
return false;
}
});
return number;
}

var value = share[field] || "";
return value ? (key || "") + window.encodeURIComponent(value) : "";
});
}
